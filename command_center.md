## 1. このアカウントは何か

**コンセプト**：共働き家族の「夜ごはん」を、時間とお金の数字で解決する保存アカ

**核メッセージ**：「家族4人分で600円以下・15分以内。それで十分うまい。」

**差別化の旗**：全投稿に必ず「○分」「家族○人分○円」の2つの数字を主役として出す。定番・簡単レシピが土台。意外性は価格の安さとスピードで出す（味の冒険はしない）。

**週テーマ制**：毎週1テーマを設定し、月〜金は同テーマで1本ずつ投稿。土曜に5本まとめカルーセル＋買い物リスト1枚を出す。テーマは食材縛り／予算縛り／調理法縛り／シーン縛りでローテ。

**ターゲットペルソナ**：
- 共働き主婦・30代中心。家族2〜4人の夕食を毎日回している
- 帰宅後に「今日何作ろう」と悩む時間も体力もない
- 安くて早くて家族が食べてくれるものを常に求めている

**伸びる指標**：保存数 ＞ いいね数。特に週まとめカルーセルと買い物リストの保存が最重要。

## 3. 中身の方針

- **共感：情報データ比率** = 2：8（情報中心）
- **メインコンテンツ**：レシピ（材料・手順・コツ・コスパ情報）。必ず「○分」「家族○人分○円」の数字を主役にする
- **分量基準**：2〜4人分（家族夕食を想定）
- **価格基準**：家族4人分で600円以下（これを超えるレシピは出さない）
- **テーマ範囲**：
  - 冷凍食材フル活用（冷凍うどん・冷凍肉・冷凍野菜）
  - レンジ・トースターだけで完結するレシピ
  - 材料3〜5つ以内
  - 調理時間15分以内（目標10分以内）
  - 定番・検証済みレシピを下敷きにする（未検証の味の冒険はしない）
- **意外性の出し方**：価格の安さ・速さ・ギャップ（お店みたいなのに手抜き）で出す。味の奇抜さで出さない
- **除外**：高級食材・調理時間15分超・特殊器具必須・家族4人分600円超のレシピ

## 4. NGルール（炎上防止・絶対）

> ⚠️ 以下を1つでも違反した投稿は必ず作り直すこと。違反したまま投稿しない。

**NG①：特定商品・メーカー名の言及禁止（通常投稿）**
- NG例：「〇〇社の△△が最高！」「××ブランドのこれを使って」
- OK例：「冷凍うどんの5食パック（どこのスーパーでも売ってるやつ）」のように一般化
- 例外：PR案件のみ可。その場合は必ず「#PR」「#広告」を明示すること

**NG②：医療・健康効果を断言する表現の禁止**
- NG例：「これを食べれば痩せる」「血圧が下がる」「美肌になる」「免疫が上がる」
- OK例：「たんぱく質が摂れる」「野菜を手軽に足せる」「食物繊維が豊富」程度の一般表現

**NG③：このアカの軸を外れたレシピの禁止**
- 家族4人分600円超・調理時間15分超・特殊器具必須のレシピは出さない
- 「時短＋安い」という軸からブレた投稿は、コンセプト崩壊につながる

**NG④：食品衛生・加熱安全の表現禁止**
- 肉類（鶏・豚・ひき肉）は「中心まで加熱」が前提。半熟や低温調理を推奨する表現は禁止
- 卵の半熟提案をする場合は「新鮮な卵を使用・自己責任で」の一言を必ず添える
- レンジ加熱の鶏むね・ひき肉レシピは加熱時間を必ず明記し、「火が通ったか確認してから」を添える
- 「子どもに食べさせても安心」「アレルギーがあっても食べられる」等の安全断言は禁止

## 5. 投稿運用

**週テーマ制の構造**：
```
月〜金：毎日1リール（その週テーマの1レシピ）
         → 1枚文字込み画像を軽量リール化（静止画＋動き）
土曜 ：まとめカルーセル投稿
         → 「今週の家族ごはん5品・合計○円・買い物リスト付き」
         → 単なる再放送にしない。「1週間の献立をこれ1枚で完結」という付加価値を必ず載せる
```

**週テーマのローテーション軸**（被り防止）：
- 食材縛り（冷凍うどん週・もやし週・豆腐週 等）
- 予算縛り（今週は全品300円台 等）
- 調理法縛り（レンジだけ週・トースターだけ週 等）
- シーン縛り（疲れた日の5分週・月曜リセット週 等）

- **目標頻度**：毎日1本（月〜金リール＋土曜カルーセル）
- **死守ライン**：週5本＋土曜まとめ（月40本ペース）
- **画像制作**：AIハイブリッド（AI料理写真＋Canvaで日本語テキスト上載せ）
- **CTA（行動喚起）**：
  - 保存促進：「今週分まとめて保存しておくと献立に困らない！」
  - フォロー促進：「毎週テーマ別に家族ごはんレシピを投稿中」
  - コメント促進：「今週のテーマリクエストはコメントへ」

**推奨ハッシュタグ（固定枠）**：
```
#時短レシピ #簡単レシピ #節約レシピ #家族ごはん #共働きごはん #レンジレシピ #冷凍食材活用 #週末作り置き #献立
```
投稿固有タグを2〜3個追加（例：#冷凍うどんアレンジ #今週の献立）

## 6. ビジュアル定義

- **全体の雰囲気**：温かみ・食欲をそそる。明るいが主張しすぎない。清潔感がある。
- **背景色**：クリーム系ベージュ（#FFF8E7〜#FFF0D0）
- **メインカラー**：ホットピンク（#FF4E8C）・ネイビー（#1A2B5E）
- **サブカラー**：グリーン（#4CAF50）・ホワイト（#FFFFFF）
- **フォント**：太め・インパクト重視（和文太字）。サイズ差で情報の階層を出す。
- **必須要素**：実際の料理写真（完成写真必須）。アイコン・チェックマーク・矢印で視線誘導。
- **必須要素（全投稿共通）**：タイトルに「○分」と「家族○人分○円」の数字を必ず入れる。価格バッジ（¥○○）をフィードサムネイルの左下に常設。
- **標準レイアウト構成（単品レシピ）**：
  - 上部：タイトル（時間・価格を目立たせる）＋完成写真
  - 中段左：材料リスト（2〜4人分、食材名＋分量）
  - 中段右：おいしくなるポイント（チェックマーク3〜4個）
  - 下段：作り方ステップ写真3枚（番号付き＋1行説明）
  - フッター：キャッチコピー＋@iemeshi_jp
- **土曜カルーセル構成**：
  - 1枚目：「今週の家族ごはん5品まとめ・合計○円」（表紙）
  - 2〜6枚目：各レシピの1枚サマリー（価格・時間・材料だけ）
  - 7枚目（最終）：今週分の買い物リスト1枚（食材名・数量・目安価格）

## 8. 困ったときの判断基準

迷ったら、この4つを順番に確認する：

1. **ペルソナに届くか？** 共働き主婦30代が「今夜これ作れる！」と思うか
2. **数字が主役か？** 「○分」と「家族○人分○円」が画像タイトルに入っているか
3. **NGを踏んでいないか？** 特定商品言及・医療断言・600円超・加熱安全の4つを全てクリアしているか
4. **軸を守っているか？** 「定番・簡単・家族向け」の土台から外れていないか

4つ全部OKなら投稿する。1つでも引っかかったら作り直す。
